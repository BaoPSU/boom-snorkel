# Boom Snorkel

Open source 3D printed articulating fume extraction arm. Attaches to a fume extractor and positions the intake snout over your work area. Fully repositionable -- loosen the M8 rod, move it, retighten.

## How it works

An M8 threaded rod runs through the entire arm. The 3D printed joints have wave-face (serrated) mating surfaces. When the rod is loose, joints rotate freely. Tighten the nut and the wave faces mesh together, locking every joint in position simultaneously.

## Bill of materials

| Part | Notes |
|---|---|
| PVC pipe | Tube sections between joints |
| M8 threaded rod | Runs full length of arm |
| M8 nuts | Tension adjustment at one end |
| 3D printed joints | Wave-face locking, printed in PLA/PETG |
| 3D printed snake adapter | Attaches original flexible snake tube to the arm |

## Prints

- `joint.stl` -- articulating joint with wave-face locking surfaces
- `snake_adapter.stl` -- adapter for standard flexible fume extractor snake tube

## Assembly

1. Thread M8 rod through all joints and PVC segments
2. Attach snake adapter at the intake end
3. Hand-tighten nut to desired stiffness
4. Loosen to reposition, retighten to lock
