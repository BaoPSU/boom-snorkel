# Improvements for Next Student

This file tracks known issues and suggested changes to hand off to the next person working on this project.

## Joint design

- **Smaller wave teeth** -- the current wave-face teeth are large enough that the joint locks in coarse increments. A version with smaller/finer teeth would allow more precise positioning. Try a few tooth pitch variants and compare how well they hold under the arm's weight.
